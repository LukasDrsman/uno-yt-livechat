# uno-ytlive-chat
## Usage
Requirements:
* sh
* python3
* linux (or macOS)
* pytchat
* pyserial
* unidecode

<br/><br/>
Install, upload and run:
```sh
git clone https://github.com/LukasDrsman/uno-yt-livechat.git
cd uno-yt-livechat
# edit and rename config-example.py to config.py
./upload
./build
yt-livechat
```
