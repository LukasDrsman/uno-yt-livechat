# uno-ytlive-chat
## Usage
#### Requirements:
* python3 with pip
  * pytchat
  * pyserial
  * unidecode
  
#### Install dependencies:
```sh
pip install pytchat pyserial unidecode --user
```
#### Install, upload and run:
```sh
git clone https://github.com/LukasDrsman/uno-yt-livechat.git
cd uno-yt-livechat
# edit and rename config-example.py to config.py
./upload
./build
yt-livechat
```
